# Getting Started with the Vue Mention Component

## Repository Description
A quick-start project that helps you integrate a Vue Mention component in Vue applications, featuring customizable popup dimensions, trigger characters, filtering options, and item sorting.

## Overview
This project provides a complete setup for working with the Syncfusion Vue Mention component. It includes practical code snippets for customizing the popup height and width, changing trigger characters, adding suffix text, modifying filter types, and sorting mention items.

## Features
- Customizable popup height and width
- Flexible trigger character configuration
- Suffix text customization
- Multiple filter type options
- Item sorting capabilities
- Vue 3 compatibility

## Project Prerequisites
Ensure you have the following installed on your machine:
- [Visual Studio Code](https://code.visualstudio.com/download) (latest version)
- [Node.js](https://nodejs.org/en/download) (latest LTS or later)

## Installation
1. Clone the `getting-started-with-the-vue-mention-component` repository
2. Open the project in Visual Studio Code
3. Install dependencies: `npm install`

## How to Run
1. Build the project
2. Run your application to view the output
